## Companion skills — obsidian-skills

obsidian-second-brain delegates **Obsidian format syntax** to the companion
[obsidian-skills](https://github.com/afidurko/obsidian-skills) package. Read
`Knowledge/Obsidian Skills Reference.md` in this vault for the index.

When writing vault files, apply these skills:

| Skill | Apply when |
|-------|------------|
| obsidian-markdown | Any `.md` note — wikilinks, callouts, frontmatter, embeds |
| obsidian-bases | `.base` files under `Bases/` |
| json-canvas | `.canvas` visualizations |
| obsidian-cli | Vault operations when Obsidian desktop is open |
| defuddle | Web URLs before ingest (clean markdown extraction) |

Skill paths: `obsidian-skills/skills/<name>/` (sibling clone beside AOA Financial).

This section was added by the obsidian-skills bridge. Do not remove — it keeps
format knowledge synchronized with obsidian-second-brain.
