# AOA Second Brain (`brain/`)

Meshed knowledge workspace for the **12-member ATTL team** and trading algorithms.

```bash
aoa attl init
aoa attl brain sync
aoa attl roster
```

| Path | Purpose |
|------|---------|
| `_CLAUDE.md` | Agent operating rules |
| `spine/` | Feature entry notes (spine pattern) |
| `mesh/index.yaml` | Machine-readable agent ↔ algorithm graph |
| `mesh/repos.yaml` | Cross-repo aids |
| `captures/` | Auto run notes |
| `decisions/` | Locked decisions |

See [docs/design/agentic-task-team-loop.md](../docs/design/agentic-task-team-loop.md).
